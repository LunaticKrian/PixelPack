## 任务背景
当前项目根目录下的 `/web` 是仿 RPG 像素游戏的物品管理系统。现有界面的色彩、布局和交互反馈过于平淡，缺乏像素风的灵魂。

## 核心设计目标
请基于“蓝白灰”主色调，重新构建一套极具沉浸感的像素风格 UI。具体要求如下（请严格遵循）：

### 1. 色彩系统（Color Palette - 蓝白灰基调）
请严格限制使用以下色板，确保视觉统一：
- **主背景（深空蓝）**：`#0B1A2D` 或 `#1A2D4C`（深邃的洞穴/地牢感）。
- **次级背景/卡片（石板灰）**：`#2A3B5C` 配合 `#E5E9F0`（浅灰白用于文字）。
- **高亮/强调（电光蓝）**：`#4A90E2` 或 `#6BB5FF`（用于按钮、选中状态、魔法特效）。
- **边框与装饰（钛白灰）**：`#8A9BB0`（用于像素描边）。
- **字体颜色**：主文字 `#F0F4F8`，辅助文字 `#9BB0C8`。
- **警告/稀有度**：允许极少量使用 `#FFD700`（金色）作为稀有物品的点缀。

### 2. 像素美学规范（Pixel Aesthetic Rules）
- **字体**：全局使用像素风格字体（如 `Press Start 2P` 或 `Pixelify Sans`），如果加载失败，回退方案必须使用等宽字体（`monospace`）并加粗。
- **边框**：**禁止使用圆角**（`border-radius: 0`）。所有容器边框必须是直角，且使用 `3px` 到 `4px` 的双层边框（外边框深色 `#0B1A2D`，内边框亮灰 `#8A9BB0`）来模拟像素游戏的描边。
- **阴影**：不要使用 CSS 模糊阴影（`blur`）。请使用 **硬边像素块阴影**（`box-shadow: 4px 4px 0px 0px #0B1A2D` 或多层阶梯阴影）。
- **图标**：物品图标必须使用像素风格的 Emoji 或 SVG 矢量像素画，避免使用过于圆滑的现代图标库。

### 3. 交互反馈（Interaction & UX）
- **悬停（Hover）**：鼠标悬停时，按钮或卡片整体向上移动 `-2px`，并加深底部硬阴影，营造“即将被拿起”的物理感。
- **点击（Active）**：点击时，按钮需有“下沉”效果（`transform: translateY(2px)`），阴影缩短。
- **选中状态**：选中的物品栏必须有一个 **扫描线或呼吸光效**（蓝白色渐变闪烁的伪元素边框），以区分当前操作项。
- **滚动条**：自定义滚动条，使其呈现像素块状的滑块。

### 4. 布局与功能分区（Layout）
请采用经典的 RPG 物品栏布局：
- **顶部状态栏**：显示“背包容量（例如：15/30）”和“金币数量（💰 1280）”，使用像素分割线。
- **左侧/上方筛选区**：按物品种类筛选（全部 | 武器 | 防具 | 消耗品 | 材料），筛选按钮样式为像素标签页。
- **核心网格区（Grid）**：物品以 **6x6** 或 **8x8** 的网格排列。每个物品卡片包含：像素图标、物品名称、品质颜色边框（白/蓝/紫/金）。
- **右侧/底部详情面板**：点击物品时，右侧弹出详情面板，展示物品描述、属性数值（ATK+5, DEF+3）和“使用/装备/丢弃”像素按钮。

### 5. 技术细节（额外加分项）
- 背景添加极淡的、缓慢滚动的像素网格线或星空点阵，增加地牢氛围感，但不影响前景可读性。
- 所有切换动作（如筛选、翻页）尽量配合简单的 CSS 关键帧动画（如淡入和上浮）。
